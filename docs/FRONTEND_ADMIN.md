# Front-end de Backoffice (Triagem)

## Objetivo
Interface simples para triagem de tickets e acompanhamento de handoff.

---

## Telas (MRP)

- Login (autenticação básica/OIDC)
- Lista de tickets (filtros: aberto, em atendimento, resolvido)
- Detalhe do ticket (histórico da conversa, marcar como resolvido)

---

## Regras

- Apenas pessoal autorizado
- Logs de auditoria para mudanças de status
